🌌 MALLORN — Astronomical Classification Challenge
📌 Overview

This repository contains my participation in the MALLORN Astronomical Classification Challenge on Kaggle.
The objective of the competition was to classify astronomical objects using observational data using machine learning and deep learning techniques.
